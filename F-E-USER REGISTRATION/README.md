# F-E LOG IN SYSTEM (React Native - Expo)

A minimal React Native (Expo) front-end for a registration/login flow.

**Preview image included in project (local path):**
`/mnt/data/Screenshot 2025-11-10 233838.png`

## Quick start
1. Install Expo CLI: `npm install -g expo-cli`
2. `cd` into this folder.
3. Run `npm install`
4. Start with `npm start` or `expo start`
5. For the Android emulator, the backend URL used in code is `http://10.0.2.2:4000` (adjust when testing on device or Expo Go).
