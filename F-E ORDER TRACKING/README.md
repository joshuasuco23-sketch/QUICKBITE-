QuickBite - My Orders (Frontend)
This is a simple React (Vite) frontend that reproduces the "My Orders" screen.

How to run:
1. Install dependencies:
   npm install
2. Start dev server:
   npm run dev
Then open the local URL shown by vite (usually http://localhost:5173).

Files:
- package.json
- public/index.html
- src/main.jsx
- src/App.jsx
- src/styles.css
- src/assets/* (placeholder images)
