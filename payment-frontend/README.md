# Payment Methods - React Frontend (Vite)

This is a minimal React (Vite) frontend that recreates the "Payment Methods" screen from your screenshot.

## How to run

1. Ensure you have Node.js (v16+) and npm installed.
2. Extract the zip and open a terminal in the project folder.
3. Run:

```bash
npm install
npm run dev
```

4. Open the local dev URL printed by Vite (usually http://localhost:5173).

Files included:
- package.json
- vite.config.js
- public/index.html
- src/main.jsx
- src/App.jsx
- src/PaymentMethods.jsx
- src/styles.css

You can customize icons and text as needed.
